# Ejercicios de JavaScript

Este repositorio está dedicado a prácticar  y consolidar mis conocimientos en JS. Aquí iré haciendo ejercicios, pruebas o pequeños proyectos para que vaya aprendiendo durante mi formación.

La idea es reforzar los fundamentos basicos y mejorar mi capacidad de resolver problemas para ganar mas soltura escribiendo código. 

Cada ejercicio es una oportunidad para poner en práctica la teoría y experimentar.
